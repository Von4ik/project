Visual Studio Code

Visual Studio Code (VS Code) is a text editor developed by Microsoft for Windows, Linux and macOS. It is positioned as a "lightweight" code editor for cross-platform development of web and cloud applications. It includes a debugger[9], tools for working with Git[10], syntax highlighting, IntelliSense[11] and tools for refactoring. It has extensive customization options: custom themes, keyboard shortcuts and configuration files. It is distributed free of charge, developed as open source software[12][13], but ready-made assemblies are distributed under a proprietary license[14]. Visual Studio Code is based on Electron[15] and is implemented through the Monaco web editor[16], developed for Visual Studio Online.
